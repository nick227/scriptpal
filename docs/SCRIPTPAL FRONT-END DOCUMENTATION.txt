## SCRIPTPAL FRONT-END DOCUMENTATION

### Overview

The front-end of ScriptPal controls the chat interface and the script editor.

### Features

The AI chat assistant widget, the script editor, and the script import widget are all implemented in the front-end.
Autosave is implemented for the script editor.
The script editor includes a minimap, a keyboard manager, and a page tracker.
The script editor can import scripts from a text file.
