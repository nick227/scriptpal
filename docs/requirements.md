Bug / Feature List

1. Must be able to title, save and retrieve multiple scripts from a script list dropdown.

2. Carriage return cycles through our format pattern, which ends by alternating Speaker <-> Dialog

3. AI generated script must user our script markdown style

4. The application can appen AI generated script to the current script at target locations.

5. The AI can change and replace sections of script.

6. The AI can discuss the script with the user.

7. The system retains script attributes and include it as AI context on a script by script basis.

8. The user can switch active script loading the script ui and switching the AI context.

9. The user can ask the AI to tell them about the script and get a valuable response.

10. Each script has it's own distinct chat history.

11. The chat history changes on script change.

12. The current active script and chat conversation persist on page load.

13. All scripts have a title page - the title page is the permanent new first page and contains title, author, date.

14. Users can minimize, re-size and reposition the chat container.

15. Carriage return always changes the line format. 

16. Shift+carriage return creates a normal new line.

17. Users can select multiple lines.

18. Users delete multi-selected lines.

19. Users can change the line format using right and left arrow keys.

20. The script auto-saves after every new line.

21. If user presses ctrl+s saves script and flashes the save icon in topbar.

22. System tracks script change history.

22. Controls bar contains buttons: undo redo - allowing user to step through script editting history.

23. Controls contains format buttons - allowing user to format current row

24. System tracks and renders pages properly when adding or deleting text.

25. Users can add chapter breaks, chapter breaks contain auto incrementing number and name of chapter.

26. Users can chat with AI through the chat interface. AI always replies, might include script edits.

27. The AI response may contain target line number to insert AI generated script.

28. AI generated script uses system line formatting convention.

29. Users can login logout and register with email. 

30. Users can go full screen that hides the top bar and navbar and uses max screen height for pages.
